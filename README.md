1. Clone the repository:

2. Install dependencies:

From the root folder, open two terminals:

```bash
# Install backend dependencies
cd backend 
npm install

# Install frontend dependencies
cd frontend 
npm install
```

3. Running the App:

From the root folder, open two terminals:

**Start the backend:**
```bash
cd backend 
npm node.js
```

**Start the frontend:**
```bash
cd frontend 
npm run dev
```
